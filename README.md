# Kawamoto - A JS Gameboy Emulator

An Electron-based GameBoy emulator with a React UI.

## Developer Setup

Setup assumes you have a recent version of Node.

1. `npm install`

## NPM Scripts

| Command | Description |
| ------- | ----------- |
| `npm start` | Launches the application in development mode. |
